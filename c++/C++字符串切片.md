# C++字符串切片

## `substr()`函数

`substr(0, 4)` 从0开始, 长度为4

## `find()`函数

`str1.find(str)`

如果str在`str1`中, 返回第一个字符的下标

如果不在`str1`中, 返回一个整数-1

```c++
initial_str='01234567';
str = 23;
int index = initial_num.find(str);
// return index = 2

no_str = abc;
int result = initial_num.find(no_str);
// return result==-1
```
